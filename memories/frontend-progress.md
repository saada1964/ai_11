### API Endpoints المتاحة:
1. Agent: /agent/invoke, /agent/invoke-stream
2. Conversations: /conversations/
3. Credit: /credit/ (codes, balance, transactions, payments)
4. Models: /models/
5. Tools: /tools/
6. Health: /health/

### التقنيات المستخدمة:
- React 18 + TypeScript
- Tailwind CSS
- React Query
- Socket.io (للـ streaming)
- i18next (عربي/إنجليزي)
- React Router
- Framer Motion
